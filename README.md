# Three.js Lab

Experiments and learning projects with [Three.js](https://threejs.org/).

## Projects

### `basic-project`

Minimal setup using an import map and Three.js from a CDN. Open `basic-project/index.html` in a browser or serve the folder with any static server.

### `vite_project`

Vite + Three.js setup for local development with hot reload.

**Requirements:** Node.js 20+

```bash
cd vite_project
nvm use 20
npm install
npm run dev
```
